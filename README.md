# calendar
Offline-first calendar app.

### 1. Launching the project 

1. Open `/dist/index.html` in latest Chrome.

### 2. Usage
    - usage details will be listed here

### 3. Development mode

1. `npm install`
2. `npm start`
3. Open http://localhost:3000 in latest Chrome.

### 4. Running tests

1. `npm test`


#### Technologies used
- Client: React, Redux, ES6
- Tests: Mocha, Chai
